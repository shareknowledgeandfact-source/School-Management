School Management System

A simple School Management System built with Python using Object-Oriented Programming (OOP) concepts and JSON for data storage. This project allows users to register students and teachers, assign grades, and store records persistently.

Features
Student Registration,
Teacher ,
Email Validation,
Add Grades to ,
Store Data in JSON File,
Prevent Duplicate Roll Numbers,
Menu-Driven Console Interface,
